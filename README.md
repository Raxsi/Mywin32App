# Windows Basic App

A simple Win32 desktop application demonstrating menus, modal dialogs, and modeless dialogs.

## Features
- Menu with File/Open, Exit
- About dialog (modal)
- Custom modeless dialog

## Build
Open the solution in Visual Studio and build.
